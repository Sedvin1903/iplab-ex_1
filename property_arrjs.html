Array Properties

i] Concat() - The concat() method concatenates (joins) two or more arrays.

ii] Join() - The join() method returns an array as a string.

iii] Splice() - The splice() method adds and/or removes array elements.

iv] Fill() - The fill() method fills specified elements in an array with a value.

v] indexof() - The indexOf() method returns the first index (position) of a specified value.

vi] isArray() - The isArray() method returns true if an object is an array, otherwise false.

vii] lastIndexOf() - The lastIndexOf() method returns the last index (position) of a specified value.

viii] copyWithin()

ix] length() - The length property sets or returns the number of elements in an array.

x] reverse() - The reverse() method reverses the order of the elements in an array.

xi] sort() - The sort() sorts the elements of an array.

xii] map() - map() creates a new array from calling a function for every array element.
