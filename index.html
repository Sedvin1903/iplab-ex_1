Regular Expressions in JavaScript

Regular Expression Object

Property/Method Description
source Returns the text of the regular expression pattern
flags Returns a string of all the flags used in the regular expression
test() Tests for a match in a string. Returns true or false.
exec() Searches for a match in a string. Returns an array of information or null if no match is found.

Modifiers

Modifier Description
i Case-insensitive matching
g Global matching (finds all matches rather than stopping after the first match)
m Multi-line matching

Brackets

Expression Description
[abc] Matches any of the characters a, b, or c
[^abc] Matches any character except a, b, or c
[0-9] Matches any digit between 0 and 9
[^0-9] Matches any non-digit character
[A-Z] Matches any uppercase letter
[a-z] Matches any lowercase letter

Metacharacters

Metacharacter Description
. Matches any single character except for newline or line terminator
\w Matches any alphanumeric character (including underscore)
\W Matches any non-alphanumeric character (excluding underscore)
\d Matches any digit character
\D Matches any non-digit character
\s Matches any whitespace character
\S Matches any non-whitespace character
\b Matches a word boundary (position between a word character and a non-word character)
\B Matches a non-word boundary (position between two word characters or two non-word characters)
\0 Find a NULL character
\n Find a new line character
\f Find a form feed character
\rFind a carriage return character
\tFind a tab character
\vFind a vertical tab character

Quantifiers

Quantifier Description
* Matches zero or more occurrences of the preceding element
+ Matches one or more occurrences of the preceding element
? Matches zero or one occurrence of the preceding element
{n} Matches exactly n occurrences of the preceding element
{n,} Matches at least n occurrences of the preceding element
{n,m} Matches at least n and at most m occurrences of the preceding element

RegExp Methods

Method Description
test() Tests for a match in a string. Returns true or false.
exec() Searches for a match in a string. Returns an array of information about the match or null if no match was found.
match() Searches for a match in a string. Returns an array of information about the match or null if no match was found.
replace() Replaces a substring in a string with a new substring.
search() Searches for a match in a string. Returns the index of the match or -1 if no match was found.

Conclusion

Regular expressions are a powerful tool for working with text data. They allow you to search for patterns in strings and perform complex text processing tasks with ease. By understanding the components of a regular expression, including the object, modifiers, brackets, metacharacters, quantifiers, and methods, you can become a more efficient and effective programmer.
